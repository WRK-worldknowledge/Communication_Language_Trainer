# Communication Language Trainer ✈️

An AI-powered training environment for aviation students to practice real-life communication scenarios with passengers.

## 🎯 Purpose
This tool allows students to:
- Practice communication skills in realistic situations
- Improve language proficiency (EN, NL, FR, DE, ES)
- Develop professional behaviour in customer interactions

## 🧠 Concept
Students interact with an AI passenger in scenario-based simulations (e.g. check-in, gate, onboard).  
The passenger responds dynamically based on the student's communication.
